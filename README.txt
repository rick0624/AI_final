在生活中，我們時常會對生活的交通壅塞感到煩躁，尤其是生活在大都市裡的我們，每天面對著車水馬龍的交通量早已見怪不怪。於是我們組選擇在Kaggle上選擇與交通相關的主題，而經搜尋過後我們找到了這個跟共享單車租借有關的問題。該資料集有⾜夠龐⼤的筆數，⽬前也尚未有很多參賽者加入可避免我們直接透過網路搜尋前⼈的⽅法，使該題⽬更有挑戰性，此外也是非常貼近⽇常⽣活的題⽬，希望透過該分析能夠展業我們在課堂上學習到的實務技術，並與⽇常應⽤結合。
至於為何交通問題會如此重要呢? 近幾年來基於交通情況與環保意識提升，空汙相關的問題也越來越受重視。而現今越來越多人選擇共享單車，但不同情況下，單車的租借量會有所差別，若錯估租借量，可能對業者甚至是政府帶來非預期的損失。所以透過AI的預測，即可使他們對各種情況單車的分配做適時調整，進而達到長久的發展。此外，假使能準確預估單車用量也有助於單車業者在成本上達到有效的控管，甚至達成經濟規模上的有效效益，使他們經營更加有成效。
